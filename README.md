# ContextVocab — 文脈で覚える単語帳

知らなかった単語を起点に、AI（プロトタイプではモック）がその単語を含む英文を生成し、多読しながら覚える新感覚の単語帳・多読 Web アプリの MVP プロトタイプです。

**公開URL:** https://soba314159.github.io/context-vocab/

## 機能
- 単語登録 & 3D フラッシュカード
- 要復習の単語を含む英文を生成するコンテキスト・リーディング（日本語訳トグル / 単語タップで意味表示＆その場登録）
- 忘却曲線ベースの間隔反復（熟練度で出現頻度が変化）
- 継続の仕掛け：育つ「マイ・アイランド」＋連続ログインの炎ストリーク
- PWA（ホーム画面に追加・オフライン対応）

## 技術
- 単一 HTML（Vanilla JS + Tailwind CSS[CDN]）。ビルド不要。
- 状態は端末内 `localStorage` にのみ保存（サーバー送信なし）。

## MVP 運用メモ
- **フィードバック導線**: ホームの「💌 感想・要望を送る」。送信先は `index.html` 内 `FEEDBACK_URL`（既定はメール。Google フォーム等に差し替え可）。
- **かんたん計測**: 起動回数を無登録カウンター（counterapi.dev）で集計（1日1端末）。個人情報・Cookie は使いません。
  - 合計の確認: https://api.counterapi.dev/v1/ctxvocab-mvp-8f3k/opens/
  - アプリ内表示: `index.html?stats` を開く
